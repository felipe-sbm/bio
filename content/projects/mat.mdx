No site você encontra:

- vídeo aulas dos conteúdos fornecidos pelo professor
- seção de comentários para alunos
- uso de boostrap nas páginas
- implementação do google ads, monetizando o site
- cronograma de eventos futuros
- programado usando next.js

No aplicativo você encontra:

- Quiz de matemática, para uso em sala de aula
- banco de dados com lista de questões
- vídeo aulas divididas por série/ano
- disponibilidade para android
- programado usando react.js

Para acessar o site, basta clicar no link abaixo:

https://matematicacomcuscuz.com.br

Para baixar o aplicativo, basta clicar no link abaixo:
(exclusivo para dispositivos android)

https://play.google.com/store/apps?hl=pt_BR

## Open Source
O projeto é open source e pode ser encontrado no GitHub neste endereço:
[GitHub](https://github.com/felipe-sbm/mat)

Caso queira baixar os arquivos em seu computador, basta entrar no terminal do VSCODE e digitar:

```
git clone https://github.com/felipe-sbm/mat.git
cd mat
```

Caso você queira ultilizar os arquivos para criar o seu prórprio projeto, basta adicionar os créditos do criador em um arquivo LICENCE ou README.
Após adicionar os créditos, você está liberado para a construção do seu projeto, tendo liberdade para edição e distribuição do seu projeto.
